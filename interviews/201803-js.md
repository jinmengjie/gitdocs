#### new 一个对象具体做了什么

1、创建一个新对象
2、将构造函数的作用域赋给新对象（this指向这个新对象）
3、执行构造函数中的代码（为这个新对象添加属性）
4、返回新对象

`如果构造函数有return一个对象，直接返回return的内容，否则返回创建的这个对象`

```javascript
    function New (f) {
/*1*/  var n = { '__proto__': f.prototype };
       return function () {
/*2、3*/    f.apply(n, arguments);
/*4*/    return n;
       };
    }
```
